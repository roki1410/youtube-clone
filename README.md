# YouTube Clone - Plataforma de Video

Una plataforma estilo YouTube completamente funcional construida con HTML, CSS y JavaScript. Todo el almacenamiento es temporal y se pierde al cerrar la pestaña.

## Características

- ✅ Página principal con miniaturas avanzadas
- ✅ Buscador profesional con autocompletado
- ✅ Página de video con reproductor personalizado
- ✅ Página de Shorts con reproducción vertical
- ✅ Diseño responsive para todos los dispositivos
- ✅ Múltiples temas (claro, oscuro, OLED, sepia)
- ✅ Favoritos temporales
- ✅ Historial temporal
- ✅ Configuración en memoria
- ✅ Comentarios simulados
- ✅ Minijuegos para videos con gameTag
